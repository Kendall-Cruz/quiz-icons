## Context

This repository was created as part of a React Native application built for academic practice.

Tech stack used:
- React Native
- Node.js
- Express
- MongoDB Atlas
- Mongoose

During development, I initially tried storing external icon URLs in the database. However, many third-party URLs were inconsistent or unavailable, which caused broken UI assets.

To solve this, I created a centralized static icon repository to ensure reliability and consistency.

This repository is meant to support the backend service of the application.
